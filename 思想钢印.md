### Files Modified
- backend/railway.json
- frontend/railway.json
- frontend/package.json
- README.md

### Description
Removed repository-root Railway config to avoid accidental root-level build detection in monorepo deployments. None.

### Database Changes
None.

### Notes
For fastest deployments, both Railway services must set correct Root Directory and watch paths (`backend/**`, `frontend/**`) in Railway dashboard. This is required to prevent cross-service rebuilds when only one side changes.
